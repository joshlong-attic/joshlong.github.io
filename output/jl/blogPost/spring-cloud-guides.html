Spring Cloud Guides

30 January 2016

Kenny and I have been hard at work on the O'Reilly Cloud Native Java book, and Phil and I have been hard at work on the updated Building Microservices with Spring Boot LiveLessons for the more in-depth introductions to Spring Boot and Spring Cloud, but I am very happy to report that we on the Spring team finally started to add guides to the Spring Guides section on service registration and discovery, Zuul-based microproxies for routing and filtering, circuit breakers, and centralized configuration! Get 'em while they're hot!
